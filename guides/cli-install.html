安装 CLI 工具 ​

QuecPanel CLI (@quec/oem-cli) 是开发面板必不可少的工具。它提供了项目创建和构建等核心功能。

TIP

如果你想了解更多关于 CLI 的信息，可以查看 CLI 工具。

环境要求 ​

  • Node.js: >= 14.0.0
  • npm or yarn

安装步骤 ​

推荐使用全局安装，以便在任何目录下都能使用 quecApp 命令。

bash
npm install -g @quec/oem-cli --registry http://192.168.23.184:4873
bash
yarn global add @quec/oem-cli --registry http://192.168.23.184:4873

注意

由于使用了私有仓库地址，请确保您的网络环境可以正常访问 http://192.168.23.184:4873。

验证安装 ​

安装完成后，可以通过查看版本号来验证安装是否成功：

bash
quecApp -v
# 输出示例：1.32.0

如果看到版本号输出，说明 CLI 已安装就绪。
