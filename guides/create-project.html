创建/拉取工程 ​

我们提供了两种方式来获取面板工程代码：使用 CLI 脚手架工具（推荐）或直接通过 Git 仓库拉取。

方式一：使用 CLI 脚手架（推荐） ​

使用 CLI 工具可以快速生成标准化的项目结构，并自动处理部分配置。

前置条件

请确保已安装 CLI 工具。如果尚未安装，请参考 安装 CLI 工具。

在终端中运行以下命令创建新项目：

bash
quecApp createPanel <ProjectName>
# 例如：
quecApp createPanel QuecPanelTemplate

交互式配置 ​

命令执行后，CLI 会引导您完成项目配置：

  1. 项目描述：输入项目的简要描述。
  2. 选择模板：通常选择 master 分支以获取稳定版本。选择 template-next 分支以获取实验性功能。
sh
? 请简单描述一下组件/面板是做什么的: 面板工程
? 请选择模版分支: template-next

配置完成后，CLI 将自动在当前目录下创建项目文件夹。

方式二：通过 Git 拉取 ​

如果您熟悉 Git 操作，也可以直接克隆远程模版（或者项目）仓库。

bash
git clone http://gitlab.quectel.com:8108/frontend/app/ReactNative/quec-panel-template.git
bash
git clone ssh://git@gitlab.quectel.com:8061/frontend/app/ReactNative/quec-panel-template.git
