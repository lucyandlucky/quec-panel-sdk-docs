面板的調試（Android / iOS） ​

前置準備

如果您還沒有創建或拉取面板工程，請先參考 面板工程的拉取 進行相關操作。

Step 1: 安裝依賴 ​

創建好面板工程後，首先需要完成依賴的安裝。建議將 npm 源切換到私有倉庫，以確保相關依賴能夠正常下載：

bash
npm config set registry http://192.168.23.184:4873

配置完成後，在項目根目錄下執行以下命令安裝依賴：

bash
yarn

Step 2: 安裝移聯萬物 App（Debug 版本） ​

在進行真機調試前，您需要安裝“移聯萬物”的 Debug 版本 App。您可以掃描下方對應的二維碼進行下載和安裝：

📷 圖片佔位：（預留兩張二維碼圖片：一張 Android，一張 iOS）

安裝完成後，請打開並註冊/登錄 App。 如果您的賬號下還沒有綁定任何設備，可以前往 QuecCloud 物聯網開發者平台 創建一台虛擬設備，然後在 App 端完成綁定。

Step 3: 配置面板調試 IP 地址 ​

(此處可以補充 Android 端配置調試 IP 地址的具體指引或操作截圖)

Step 4: 運行面板 ​

完成上述所有準備工作後，就可以啓動面板項目的本地開發服務了。在項目根目錄下運行：

bash
yarn start

啓動成功後，在 App 上點擊該設備進入面板，即可實時預覽和調試面板效果。
