創建/拉取工程 ​

我們提供了兩種方式來獲取面板工程代碼：使用 CLI 腳手架工具（推薦）或直接通過 Git 倉庫拉取。

方式一：使用 CLI 腳手架（推薦） ​

使用 CLI 工具可以快速生成標準化的項目結構，並自動處理部分配置。

前置條件

請確保已安裝 CLI 工具。如果尚未安裝，請參考 安裝 CLI 工具。

在終端中運行以下命令創建新項目：

bash
quecApp createPanel <ProjectName>
# 例如：
quecApp createPanel QuecPanelTemplate

交互式配置 ​

命令執行後，CLI 會引導您完成項目配置：

  1. 項目描述：輸入項目的簡要描述。
  2. 選擇模板：通常選擇 master 分支以獲取穩定版本。選擇 template-next 分支以獲取實驗性功能。
sh
? 請簡單描述一下組件/面板是做什麼的: 面板工程
? 請選擇模版分支: template-next

配置完成後，CLI 將自動在当前目錄下創建項目文件夾。

方式二：通過 Git 拉取 ​

如果您熟悉 Git 操作，也可以直接克隆遠程模版倉庫。

bash
git clone http://gitlab.quectel.com:8108/frontend/app/ReactNative/quec-panel-template.git
bash
git clone ssh://git@gitlab.quectel.com:8061/frontend/app/ReactNative/quec-panel-template.git
