面板的多语言配置 ​

前置准备

此步骤需要使用脚手架工具，如果您还没有安装，请先参考 CLI 安装。

1. 添加多语言配置 ​

在面板工程中，可以通过修改 src/i18n/locales/zh.ts 文件来添加多语言配置：

ts
// src/i18n/locales/zh.ts
export default {
  panel: '面板',
};

2. 在组件中使用 ​

配置完成后，可以在组件中直接引入 i18n 函数进行使用：

tsx
import React from 'react';
import { Text } from 'react-native';
import i18n from '../../i18n/i18n'; // 请根据实际文件的相对路径引入

export default function App() {
  return (
    <Text>{i18n('panel')}</Text>
  );
}

3. 生成多语言表格 ​

完成开发后，如果需要将词条交由相关人员进行统一翻译，可以运行以下命令生成多语言 Excel 表格：

bash
quecApp genI18n

命令执行成功后，会在项目目录下生成 Excel 表格文件，默认路径为： src/i18n/temp/i18n_output.xlsx

TIP

如果您想了解更多关于 quecApp genI18n 命令的参数和用法，可以查看 CLI 工具。
