NumberTslModel ​

NumberTSLModel 表示数值类型的物模型数据。它继承自基础的长连接数据点模型 TSLModel，通常用于表示设备的温度、湿度、电压等连续变化的数值。

属性说明 ​

字段名类型说明
attributeValuenumber | undefined当前数值。未获取到时为 undefined
minnumber允许的最小值边界（默认 -4294967295）
maxnumber允许的最大值边界（默认 4294967295）
stepnumber步长限制
unitstring单位符号（例如 ℃, %）
decimalCountnumber步长的小数位数（精度）
dataTypestring数据类型标识（如 Double, Float 等）
specsArray<SpecsNumber>原始的数据规范数组

常用方法 ​

除了直接访问属性，实例还提供了便捷的 Getter 方法：

方法签名返回类型说明
getUnit()string获取单位字符串
getStep()number获取步长数值
getMinAndMax(){ min: number; max: number }获取最大和最小值对象
getDecimalCount()number根据当前浮点或双精度类型和步长计算小数精度位数

使用示例 ​

可以通过 useModel 获取物模型实例后，调用其实例方法来获取特定的规格数据。

获取单位 ​

通常用于在 UI 上展示数值的单位后缀。既可以通过属性直接获取，也可以通过方法获取。

tsx
const temperatureModel = models.temperature;

// 方式 1：直接获取属性
const unit1 = temperatureModel?.unit;

// 方式 2：通过方法获取（安全兜底，如果没有配置 unit 则返回空字符串）
const unit2 = temperatureModel?.getUnit();

console.log(`单位 (属性):`, unit1); // 输出示例: ℃
console.log(`单位 (方法):`, unit2);

获取最大值和最小值 ​

通常用于限制滑块滑动范围，或者校验用户下发输入的合法范围。

tsx
const temperatureModel = models.temperature;

// 方式 1：直接获取属性
const min = temperatureModel?.min;
const max = temperatureModel?.max;

// 方式 2：通过方法获取（包含默认边界值 -4294967295 ~ 4294967295）
const minAndMax = temperatureModel?.getMinAndMax();

console.log('最小值 (属性):', min);
console.log('最大值 (属性):', max);
console.log('通过方法获取的边界:', minAndMax?.min, minAndMax?.max);

获取步长 ​

通常用于限制调节器（如加减按钮或分段滑块）的步长值。

tsx
const temperatureModel = models.temperature;

// 方式 1：直接获取属性
const step1 = temperatureModel?.step;

// 方式 2：通过方法获取（确保返回数字类型）
const step2 = temperatureModel?.getStep();

console.log('步长 (属性):', step1);
console.log('步长 (方法):', step2);

获取小数位 ​

根据数据类型（Float/Double）和步长定义推算出当前数据允许的小数精度，常用来格式化展示文本。

tsx
const temperatureModel = models.temperature;

// 方式 1：直接获取属性（在实例化时已计算解析）
const decimalCountProp = temperatureModel?.decimalCount;

// 方式 2：通过方法实时获取
const decimalCountMethod = temperatureModel?.getDecimalCount();

console.log('小数精度位数 (属性):', decimalCountProp);
console.log('小数精度位数 (方法):', decimalCountMethod);

// 结合使用场景：格式化展示
const currentVal = temperatureModel?.attributeValue;
const displayValue = currentVal?.toFixed(decimalCountProp ?? 0);
