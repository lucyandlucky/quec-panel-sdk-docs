EnumTslModel ​

EnumTSLModel 表示枚举类型的物模型数据。它继承自基础的长连接数据点模型 TSLModel，通常用于表示设备的工作模式、风速档位等有限个固定选项的功能。

属性说明 ​

字段名类型说明
attributeValuestring当前枚举的 value 值（注意通常是字符串类型的数字字面量，如 "2"）
specsArray<SpecsGeneral>原始的枚举规范数组，包含对象格式的各枚举定义（如 [{value: "0", name: "自动", dataType: "ENUM"}, ...]）
valueName{ [key: string]: string }键值对映射字典，方便直接通过 value 查找对应的 name 文字（如 {"0": "自动", "1": "制冷"}）
dataTypestring数据类型标识，固定为 ENUM

SpecsGeneral （通用数据定义）的具体结构如下

ts
interface SpecsGeneral {
  id?: number;          // 属性 ID
  value: string;        // 枚举选项的数据值（如 "1"）
  name: string;         // 枚举选项的显示名称（如 "制冷"）
  dataType: string;     // 数据类型（如 "ENUM"）
}

常用方法 ​

除了直接访问属性，实例还提供了便捷的 Getter 方法，方便快速获取中文化名称展示：

方法签名返回类型说明
getValueItemName(value?: string)string | undefined根据传入的枚举 value 获取对应的中文名称，找不到返回 '--'
getCurrentName()string | undefined直接获取当前 attributeValue 对应的中文名称，找不到返回 '--'
getCurrentItem()SpecsGeneral | undefined直接获取当前 attributeValue 对应在 specs 里的完整对象

使用示例 ​

在业务场景中，枚举类型常被用来渲染底部弹窗选择器（Bottom Sheet Picker）或单选按钮组（Radio Group）。以下列出常见操作的数据获取方式。

获取当前选中的枚举项名称 ​

通常用于页面上展示设备当前状态（例如显示“当前模式：制热”）。

tsx
const workModeModel = models.mode;

// 方式 1：利用映射字典直接通过属性查找
const currentVal = workModeModel?.attributeValue;
const nameProp = workModeModel?.valueName?.[currentVal];

// 方式 2：直接调用封装好的方法（推荐，自带 '--' 兜底）
const nameMethod = workModeModel?.getCurrentName();

console.log('当前模式名称 (属性查找):', nameProp);     // 输出示例: 制热
console.log('当前模式名称 (直接方法):', nameMethod);   // 输出示例: 制热

获取所有的可用枚举列表 ​

常用于渲染切换模式的 UI 列表。

tsx
const workModeModel = models.mode;

// 直接获取规范定义的数组
const options = workModeModel?.specs || [];

// 渲染为 UI 列表的常见伪代码示例
options.map(option => {
  console.log(`选项名称: ${option.name}, 选项下发值: ${option.value}`);
  /*
    输出示例：
    选项名称: 自动, 选项下发值: 0
    选项名称: 制冷, 选项下发值: 1
    ...
  */
});

获取当前选中的完整枚举对象 ​

不仅需要选中的名称，还需要该选项的其他完整定义信息时使用。可以通过 getCurrentItem() 直接获取当前选中的完整 SpecsGeneral 对象。

tsx
const workModeModel = models.mode;

// 直接获取当前选中项的完整信息对象
const currentItem = workModeModel?.getCurrentItem();

console.log('当前选中项的 value:', currentItem?.value); // 输出示例: 2
console.log('当前选中项的 name:', currentItem?.name);   // 输出示例: 制热
console.log('当前选中项的数据类型:', currentItem?.dataType); // 输出示例: ENUM
