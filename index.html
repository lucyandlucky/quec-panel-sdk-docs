业务组件

ModelKit

专为面板模型设计的基础架构，大幅降低业务搭建成本。

DeviceKit

聚合设备状态解析与 UI 展示展示的高级可视化组件。

AiKit

开箱即用、支持流式输出的 AI 智能交互 UI 模块。

核心 API

DeviceService

提供设备控制相关接口，快速下发指令并监听设备状态。

CloudTimerService

封装云端定时功能，便捷管理和同步设备定时任务。

RequestService

统一处理面板网络请求，内置拦截、异常处理与数据交互。

StorageService

面板本地缓存存取，提供安全可靠的数据持久化能力。

UserService

轻松获取并管理当前登录用户的状态与相关信息。

AiService

封装大语言模型对话等常用的 AI 基础交互能力。
